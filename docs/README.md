모든 에이전트/기여자는 작업 전 아래를 반드시 수행합니다.

3. 작업 브랜치에서 `npm run check` + `npm run build` 통과 확인
4. push 후 로컬 `docs/AGENT_WATCH_LOG.md` 완료 기록
5. `main` 머지 작업을 수행한 경우, `main`에서 `npm run check` + `npm run build` 재검증 후 unified 로그에 통합 요약 반영
